This is my testing site for some online projects I'm working on. The site is based on OS.JS, an open-source JavaScript Web Desktop by Anders Evanrud [1] . It has a fully-fledged window manager, Application APIs, GUI toolkits and filesystem abstraction.

Project: The Chamber Guitar

The Chamber Guitar is a gig book of trios for flute, violin, and guitar. My goal is to put together enough music for a 3-hour trio gig and make it available as a free download. Vol. 1 comprises the first 'set' - about 45 minutes of music - and contains works by Boismortier, Loeillet, Handel, and Mozart.

The scores were engraved with Lilypond, the award-winning open-source music engraving software [2]. The entire project - source code, PDFs, and videos - was created 'in the cloud' on Linux-based servers using OMET (see below).

Select Menu-> Projects-> Chamber Guitar to view, download, or print the scores.


Project: OMET Sandbox

OMET (Online Music Engraving Tools) is a project to develop an online Lilypond engraving system. I've been working on it on-and-off since 2010; it's the system I use to engrave my own scores. OMET is currently being redesigned to make more use of modern NodeJS components in OS.JS.

Select Menu->Projects-> OMET Sandbox for a demo. The Sandbox includes some examples for you to experiment with. For mobile devices, a separate keyboard that's better suited for Lilypond music note entry is included.
